《“书库”并行活动参与总则》

前言：
欢迎加入冠恩书屋“书库”并行活动。本总则旨在保障活动长期稳定运营，明确参与者的权利与义务，维护诚信、友善的阅读环境。

一、 重要更新与费用说明

参与门槛：为汇聚真正热爱阅读、珍视承诺的同路人，并保障活动长期运营，每位新参与者需支付一次性 9.9元 的报名费。

费用用途：此费用将用于覆盖部分运营成本，包括但不限于：首次邮寄邮费补贴、漂流手帐印制、系统维护及信用管理等。活动本身为非营利性质。

费用效力：支付成功后，您将获得参与 【自由借阅】 与 【书籍漂流】 双模式的永久资格（信用良好前提下），无需为单次活动再付费。

二、 活动总览

本平台并行运营两种书籍共享模式：

【自由借阅】：从书库中自由选书，独享阅读，到期寄回管理员。流程：您 ←→ 管理员。

【书籍漂流】：加入一本书的流动旅程，阅读后传递给下一位读者，并在漂流手帐中留言。流程：管理员 → 读者A → 读者B → ...。

支付报名费后，您可自由选择或同时参与两种模式。

三、 通用核心规则（违约将受严惩）

1. 爱护书籍（卫生是底线）：

  • 阅读时保持手部清洁，严禁沾染食物油渍、液体、污迹。
  • 禁止任何涂画、折角、撕页、浸泡等损坏行为。
  • 收到书时，须在24小时内检查并反馈破损情况，否则视为完好接收。阅读期间造成任何新损伤，必须立即主动告知管理员。

2. 严守时效：

  • 单次持有书籍周期为 30天。如需延期，必须提前3天申请并获批准。
  • 【漂流模式】 读者需在读完后的 7天 内完成传递给下一位读者。

3. 违约与重罚：

本书库所有书籍均附有不可替代的社群价值与运营价值。一旦发生违约，您需赔偿的不仅是书本本身，更是对社群信任的损害。因此，我们设定明确的赔偿标准：

  • 书籍基准价值：每本书以其市场定价或折旧估值计算（通常为20-100元不等）。
  • 附加价值：为维护活动秩序、弥补管理成本及对社群信任造成的损害，每本书设有 500元人民币的固定附加价值。此价值包含但不限于：活动运营管理投入、信用体系维护成本、其他参与者信任损失及追责所需投入的精力与资源。
  • 赔偿总额 = 书籍基准价值 + 500元人民币附加价值

4. 具体处罚阶梯：

● 严重违规（超时15天以上、损坏不报、消极传递）：

  • 处罚：暂停资格180天。
  • 赔偿：须在7日内，按 “书籍基准价值” 购置同版本全新正版书籍寄回，或支付等额赔偿。

● 重大违规（黑名单行为）（书籍丢失、恶意侵占、失联逃逸、严重损毁导致书籍无法继续使用）：

  • 处罚：永久封禁，全网实名公示。
  • 赔偿：须在7日内，支付 “书籍基准价值 + 500元人民币附加价值” 的全额赔偿。例如：一本定价50元的书，赔偿总额为550元。

法律追诉：对于涉及“失联逃逸”或恶意侵占，且拒绝赔偿的，我方将依据《中华人民共和国民法典》等相关法律，保留向违约方住所地或活动主办方所在地人民法院提起诉讼的权利，并有权将全部证据（包含本协议、沟通记录、身份信息等）提交司法机关处理。

四、 【自由借阅】专属细则

流程：选书 → 管理员寄出（我们付邮）→ 您阅读 → 您寄回（您付邮）。

特点：私密阅读，责任闭环，随时可借下一本。

五、 【书籍漂流】专属细则

流程：申请排队 → 管理员寄出书与手帐（我们付邮）→ 您阅读并留言 → 您寄给下一位读者（您付邮）→ 循环。

核心：书籍与漂流手帐必须一同传递，手帐是公共记忆体，请用心书写。

六、 参与资格与承诺

支付9.9元报名费即视为已阅读并同意本规则所有条款。

您的信用记录在双模式中通用。信用不良者，资格将被冻结或终止，报名费不予退还。

管理员保留对规则的最终解释权。

我已支付报名费，并郑重承诺：我将以诚信为本，严格遵守以上全部规则，爱护书籍，按时传递，共同维护这个宝贵的阅读社群。
